# Список складов для синхронизации (МойСклад)
# ================================================
#
# ВАЖНО: Указывайте ID (UUID) склада из МойСклад
# 
# Как найти ID склада:
# 1. Откройте МойСклад → Справочники → Склады
# 2. Откройте нужный склад
# 3. В адресной строке браузера найдите UUID склада
#    Пример URL: https://online.moysklad.ru/app/#good/edit?id=12345678-1234-1234-1234-123456789abc
#    ID склада: 12345678-1234-1234-1234-123456789abc
#
# Формат: один ID на строку, без кавычек
# Строки, начинающиеся с #, игнорируются
#
# ================================================

# Пример (замените на реальные ID):
# 12345678-1234-1234-1234-123456789abc
# 87654321-4321-4321-4321-cba987654321
